This paper describes the TMU system submitted to SemEval-2025 Task 11 (Bridging the Gap in Text-Based Emotion Detection). The system uses transfer learning with pre-trained language models to address multi-label emotion detection across diverse languages, achieving competitive performance in the shared task evaluation.
