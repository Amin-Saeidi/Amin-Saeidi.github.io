**Spring 2023 | Sharif University of Technology**

Implemented a complete single-cell RNA sequencing (scRNA-seq) analysis workflow using Python and Scanpy.

**Pipeline Stages:**
- **Preprocessing:** Quality control, normalization, batch correction, and highly variable gene selection.
- **Clustering:** Dimensionality reduction (PCA, UMAP), graph-based clustering.
- **Annotation:** Marker gene identification and cluster annotation using known cell type markers.

**Technologies:** Python, Scanpy, AnnData, pandas, matplotlib, seaborn
