**Spring 2023 | Sharif University of Technology**

Built a full MLOps pipeline for cell membrane segmentation in biomedical images (ISBI dataset).

**Key Contributions:**
- Preprocessed ISBI biomedical image data for deep learning.
- Fine-tuned ResNet and VGG16 for binary image classification.
- Developed a U-Net architecture for pixel-level semantic segmentation.
- Containerized the solution with Docker for reproducible deployment.
- Used MLflow for experiment tracking and model monitoring.
- Implemented an automated CI/CD pipeline.

**Technologies:** Python, Keras, TensorFlow, Docker, MLflow, U-Net
